## 低代码分类
 模型驱动、组装式、生成式的低代码平台；java/react技术栈；

## 特点能力
1.  简单：学习成本极低，和原生开发一致的体验
2.  高效：让后端做全栈开发，1人可顶4个人的开发效能
3.  目标：做研发热爱的零代码平台

## 真开源?
是的,前后端、底层核心和脚手架应用均开放，无黑盒代码。
